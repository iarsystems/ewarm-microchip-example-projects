DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for the Microchip MEC1322 EVB ASSY 6703 REV A.

  This is a simple demo to show basic code to blink LEDS on MEC1322.

COMPATIBILITY
=============
  The example project is compatible with Microchip MEC1322 EVB ASSY 6703 REV A.
  By default, the project is configured to use the I-Jet SWD interface.

CONFIGURATIONS
===============
  RAM Debug:
    This configuration is intended for debugging in the RAM memory.

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
