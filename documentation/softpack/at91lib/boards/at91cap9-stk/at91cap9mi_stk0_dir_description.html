at91cap9-stk
 
Purpose
Definition and functions for using AT91CAP9-related features, such has PIO pins, memories, etc.

Usage
  1. The code for booting the board is provided by board_cstartup.S and board_lowlevel.c.
  2. For using board PIOs, board characteristics (clock, etc.) and external components, see board.h.
  3. For manipulating memories (remapping, SDRAM, etc.), see board_memories.h.
