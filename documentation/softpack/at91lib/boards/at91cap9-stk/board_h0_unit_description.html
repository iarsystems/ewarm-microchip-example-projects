Purpose
Definition of AT91CAP9-STK characteristics, AT91CAP9-dependant PIOs and external components interfacing.

Usage
  1. For operating frequency information, see "AT91CAP9-STK - Operating frequencies".
  2. For using portable PIO definitions, see "AT91CAP9-STK - PIO definitions".
  3. Several USB definitions are included here (see "AT91CAP9-STK - USB device").
  4. For external components definitions, see "AT91CAP9-STK - External components".
  5. For memory-related definitions, see "AT91CAP9-STK - Memories".
