Purpose
This directory contains an API for configuring the Omnivision OV9655 captor.

Contents
Driver for Omnivision OV9655 captor. The Omnivision OV9655 image sensors are low voltage CMOS devices that provide the full functionnality of a single SXGA (1280x1024) camera and image processor in a small footprint package. The OV9655 provides full-frame, sub-sampled, scaled or windowed 8-bit/10-bit images in a wide range of formats, controlled through the TWI interface.
