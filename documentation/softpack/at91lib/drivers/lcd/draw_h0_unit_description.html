Description
Purpose
Provides simple drawing function to use with the LCD.

Usage
  1. Use LCDD_Fill to fill the LCD buffer with a specific color.
  2. Draw a pixel on the screen at the specified coordinates using LCDD_DrawPixel.
  3. Draw a rectangle with LCDD_DrawRectangle.
  4. Draw a string on the LCD with LCDD_DrawString.
