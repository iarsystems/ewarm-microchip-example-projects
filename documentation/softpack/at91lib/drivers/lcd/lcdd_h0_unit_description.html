Description
Purpose
Simple driver for the LCD, which takes care of the initialization.

Usage
  1. Call LCDD_Initialize to configure the LCD controller.
  2. Set the buffer displayed by the LCD with LCDD_DisplayBuffer if using peripheral LCDC, or start displaying with LCDD_Start if using HX8347.
