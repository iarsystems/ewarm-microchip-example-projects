Description
Copies the data from a whole block to another block on a nandflash.
Both blocks must be LIVE. Returns 0 if successful; otherwise returns NandCommon_ERROR_WRONGSTATUS if at least one of the blocks is not free, or a NandCommon_ERROR_xxx code.
