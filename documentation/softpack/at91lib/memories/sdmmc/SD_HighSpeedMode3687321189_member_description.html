SD_HighSpeedMode
Switch the SD/MMC card to High-Speed mode.
 
Syntax / parameters
unsigned char SD_HighSpeedMode (SdCard *pSd, unsigned char cardHsMode)

Name Type Default Description
pSd SdCard *
  Pointer to a SD card driver instance.
cardHsMode unsigned char
   
 
Return value unsigned char
   current mode is hsMode is 0xFF; error code if hsMode is 0 or 1.
Description
Switch the SD/MMC card to High-Speed mode.
pSd->transSpeed will change to new speed limit. Invoke MCI_SetSpeed() and MCI_EnableHsMode() to change MCI timing after. For SD/MMC card, the speed mode will not change back until another init.
