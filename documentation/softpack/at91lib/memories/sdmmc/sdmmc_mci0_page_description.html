Description
sdmmc_mci.h::sdmmc_mci
 
Purpose
Implementation for sdcard sd mode physical layer driver. Supply a set of sdcard sd mode's interface.

Usage
  1. SD_Init: Run the SDcard initialization sequence
  2. SD_MCI_Init : Run the SDcard SD Mode initialization sequence
  3. SD_Stop: Stop the SDcard by sending Cmd12
  4. SD_ReadBlock : Read blocks of data
  5. SD_WriteBlock : Write blocks of data
  6. Cmd0 : Resets all cards to idle state
  7. Cmd1 : MMC send operation condition command
  8. Cmd2 : Asks any card to send the CID numbers on the CMD line
  9. Cmd3 : Ask the card to publish a new relative address
  10. Cmd7 : Command toggles a card between the stand-by and transfer states or between the programming and disconnect states
  11. Cmd8 : Sends SD Memory Card interface condition, which includes host supply voltage information and asks the card whether card supports voltage
  12. Cmd9 : Addressed card sends its card-specific data (CSD) on the CMD line
  13. Cmd12 : Forces the card to stop transmission
  14. Cmd13 : Addressed card sends its status register
  15. Cmd16 : Set block length
  16. Cmd18 : Read multiple blocks
  17. Cmd25 : Write multiple blocks
  18. Cmd55 : App command, should be sent before application specific command
  19. Acmd6 : Defines the data bus width
  20. Acmd41 : Asks to all cards to send their operations conditions
  21. CmdEMMC8 : Sends eMMC EXT_CSD command
  22. CmdEMMC6 : Switches the mode of operation of the selected card or modifies the EXT_CSD registers
