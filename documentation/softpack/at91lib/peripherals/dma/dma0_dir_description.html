dma
 
Purpose
Interface for configuration the DMA controller(DMAC).

Usage
  1. Enable or disable the a DMAC controller with DMA_Enable() and or DMA_Disable().
  2. Enable or disable Dma interrupt using DMA_EnableIt() or DMA_DisableIt().
  3. Get Dma interrupt status by DMA_GetStatus().
  4. Enable or disable specified Dma channel with DMA_EnableChannel() or DMA_DisableChannel().
  5. Get Dma channel status by DMA_GetChannelStatus().
  6. Configure source and/or destination start address with DMA_SetSourceAddr() and/or DMA_SetDestAddr().
  7. Set Dma descriptor address using DMA_SetDescriptorAddr().
  8. Set source transfer buffer size with DMA_SetSourceBufferSize().
  9. Configure source and/or destination transfer mode with DMA_SetSourceBufferMode() and/or DMA_SetDestBufferMode().
  10. Configure source and/or destination Picture-In-Picutre mode with DMA_SPIPconfiguration() and/or DMA_DPIPconfiguration().
