Description
Purpose
Interface for configuration the Image Sensor Interface (ISI) peripheral.

Usage
  1. ISI_Init: initialize ISI with default parameters
  2. ISI_EnableInterrupt: enable one or more interrupts
  3. ISI_DisableInterrupt: disable one or more interrupts
  4. ISI_Enable: enable isi module
  5. ISI_Disable: disable isi module
  6. ISI_CodecPathFull: enable codec path
  7. ISI_SetFrame: set frame rate
  8. ISI_BytesForOnePixel: return number of byte for one pixel
  9. ISI_StatusRegister: return ISI status register
  10. ISI_Reset: make a software reset
