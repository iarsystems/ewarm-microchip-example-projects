MCI_SetSpeed
Configure the MCI CLKDIV in the MCI_MR register.
 
Syntax / parameters
unsigned int MCI_SetSpeed (Mci *pMci, unsigned int mciSpeed, unsigned int mciLimit, unsigned int mck)

Name Type Default Description
pMci Mci *
  Pointer to the low level MCI driver.
mciSpeed unsigned int
  MCI clock speed in Hz.
mciLimit unsigned int
  MCI clock limit in Hz, if not limit, set mciLimit to zero.
mck unsigned int
  MCK to generate MCI Clock, in Hz
 
Return value unsigned int
   The actual speed used, 0 for fail.
Description
Configure the MCI CLKDIV in the MCI_MR register.
The max. for MCI clock is MCK/2 and corresponds to CLKDIV = 0
