RTC_SetTimeAlarm
Sets a time alarm on the RTC.
 
Syntax / parameters
int RTC_SetTimeAlarm (unsigned char *pHour, unsigned char *pMinute, unsigned char *pSecond)

Name Type Default Description
pHour unsigned char *
  If not null, the time alarm will hour-match this value.
pMinute unsigned char *
  If not null, the time alarm will minute-match this value.
pSecond unsigned char *
  If not null, the time alarm will second-match this value.
 
Return value int
   0 success, 1 fail to set
Description
Sets a time alarm on the RTC.
The match is performed only on the provided variables; setting all pointers to 0 disables the time alarm. Note: in AM/PM mode, the hour value must have bit 7 set for PM, cleared for AM (as expected in the time registers).
