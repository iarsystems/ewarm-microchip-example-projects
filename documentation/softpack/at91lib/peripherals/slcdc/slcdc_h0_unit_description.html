Description
Purpose
Interface for configuration the Segment LCD Controller (SLCDC) peripheral.

Usage
  1. Initializes the Segment LCD controller using SLCDC_Configure().
  2. Clears the SLCD display buffer using SLCDC_Clear().
  3. Enable & disable SLCD controller using SLCDC_Enable() and SLCDC_Disable().
  4. Enables & disable the SLCD low power mode using SLCDC_EnableLowPowerMode () and SLCDC_DisableLowPowerMode().
  5. Adjusts the frame frequency using SLCDC_SetFrameFreq().
  6. Sets the display mode (normal/force off/force on/blinking) using SLCDC_SetDisplayMode().
  7. Adjusts the display blinking frequency using SLCDC_SetBlinkFreq().
  8. Enables & disable the selected SLCDC interrupt sources using SLCDC_EnableInterrupts() and SLCDC_DisableInterrupts().
