Description Overview
Methods Brief description
 TDES_Configure Configures the triple-DES peripheral to cipher/decipher, use single-DES or triple-DES, use two or three keys (when in triple-DES mode), start manually, automatically or via the PDC and use the given operating mode (ECB, CBC, CFB or OFB).
 TDES_GetOutputData Stores the output data from the last TDES operation into the given 64-bits buffers.
 TDES_GetStatus Returns the current status register value of the TDES peripheral.
 TDES_SetInputBuffer Sets the input data buffer to encrypt/decrypt when in PDC mode.
 TDES_SetInputData Sets the input data to encrypt/decrypt using TDES.
 TDES_SetKeys Sets the 64-bits keys (one, two or three depending on the configuration) that shall be used by the TDES algorithm.
 TDES_SetOutputBuffer Sets the output buffer which will receive the encrypted/decrypted data when using the PDC.
 TDES_SetVector Sets the initialization vector to use when the TDES algorithm is configured in a chained block mode (CBC, CFB or OFB).
 TDES_Start Starts the encryption or decryption process if the TDES peripheral is configured in manual or PDC mode.
