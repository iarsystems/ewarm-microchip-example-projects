Description
Purpose
Firmware encryption using AES hardware acceleration This code is based on the AES reference implementation published by Paulo Barreto and Vincent Rijmen.

Usage
  1. aes_hard_init: Initialize AES hardware
  2. aes_hard_init_CBC: for the CBC mode
  3. aes_hard_init_CTR: for the CTR mode
  4. aes_hard_init_ECB: for the ECB mode
  5. aes_hard_cleanup: Cleans up AES
  6. aes_hard_decrypt: Decrypts a variable-length cipher text
