Description Overview  
Description
Purpose
Firmware encryption using AES reference implementation

Usage
  1. aes_ref_init: Initialize AES hardware
  2. aes_ref_init_CBC: for the CBC mode
  3. aes_ref_init_ECB: for the CTR mode
  4. aes_ref_init_CTR: for the ECB mode
  5. aes_ref_cleanup: Cleans up AES
  6. aes_ref_decrypt: Decrypts a variable-length cipher text
  7. aes_ref_decrypt_CBC: for CBC mode
  8. aes_ref_decrypt_ECB: for ECB mode
  9. aes_ref_decrypt_CTR: for CTR mode
