Description
Purpose
Firmware encryption using TDES hardware acceleration

Usage
  1. tdes_hard_init: Initialize TDES hardware
  2. tdes_hard_init_CBC: for the CBC mode
  3. tdes_hard_init_ECB: for the CTR mode
  4. tdes_hard_init_CTR: for the ECB mode
  5. tdes_hard_cleanup: Cleans up DES
  6. tdes_hard_decrypt: Decrypts a variable-length cipher text
  7. tdes_hard_decrypt_CBC: for CBC mode
  8. tdes_hard_decrypt_ECB: for ECB mode
  9. tdes_hard_decrypt_CTR: for CTR mode
