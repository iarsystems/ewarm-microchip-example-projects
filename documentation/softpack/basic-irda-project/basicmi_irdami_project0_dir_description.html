basic-irda-project
 
Purpose
This example demonstrates the IrDA (Infrared Data Association) on AT91 microcontrollers.

Requirements
This example can be used on AT91SAM7L-EK.

Description
The provided program uses the USART in IrDA mode for transmit and receive one octet. The use of two boards are required for this test. The program receive one octet, increment it, and send it on the IrDA chanel.

Usage
  1. Build the program and download it inside the evaluation board. Please refer to the SAM-BA User Guide, the GNU-Based Software Development application note or to the IAR EWARM User Guide, depending on your chosen solution.
  2. download the program inside the second evaluation board.
  3. Connect a serial cable to the DEBUG port on the evaluation kit. It will most often be labeled "SERIAL DEBUG port".
  4. On the computer, open and configure a terminal application (e.g. HyperTerminal on Microsoft Windows) with these settings:
    • 115200 bauds
    • 8 data bits
    • No parity
    • 1 stop bit
    • Hardware flow control (RTS/CTS)
  5. Start the application by pressing the wake-up button on the two boards.
  6. The following traces shall appear on the terminal:
