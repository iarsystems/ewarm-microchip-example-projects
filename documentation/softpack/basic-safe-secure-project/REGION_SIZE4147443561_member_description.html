Syntax / parameters
define REGION_SIZE (((MEDIA_BOOT_SIZE-BOOTROMSIZE-MEDIA_BOOT_PAGE_SIZE)/2)&0xFFFFFF00)

Description
Size of one region Note that REGION_SIZE should be equal to (flash size - bootloader size - page size) / 2 One page is reserved to store information about regions Signatures, length of firmware, etc.
Ex: with SAM7XC256, BOOTROMSIZE (0x40000 - (32x1024) - 0x100) / 2 = 0x1BF00 Regions must be aligned on page and multiple of MEMORY_PAGE_SIZE
