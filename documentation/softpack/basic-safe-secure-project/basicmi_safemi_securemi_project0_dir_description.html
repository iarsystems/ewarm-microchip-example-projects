Purpose
The Basic Safe and Secure Project demonstrate how to implement a secure bootloader.

You can find following information depends on your needs:
  • Appllication note: Safe and Secure Bootloader Implementation
  • Appllication note: Safe and Secure Firmware Upgrade for AT91SAM Microcontrollers

  • Brian Gladman (http://fp.gladman.plus.com/cryptography_technology/index.htm)

Description
!!!Usage

  1. Build the program and download it inside the evaluation board. Please refer to the SAM-BA User Guide, the GNU-Based Software Development application note or to the IAR EWARM User Guide, depending on your chosen solution.
  2. On the computer, open and configure a terminal application (e.g. HyperTerminal on Microsoft Windows) with these settings:
    • 115200 bauds
    • 8 bits of data
    • No parity
    • 1 stop bit
    • No flow control Connect it to the DBGU connector
  3. Start the application.
  4. In the terminal window, the following text should appear: 
    -- USB Device Mass Storage Project xxx --
-- AT91xxxxxx-xx
-- Compiled: xxx xx xxxx xx:xx:xx --
  5. On the computer, open and configure a second terminal application (e.g. HyperTerminal on Microsoft Windows) with these settings:
    • 115200 bauds
    • 8 bits of data
    • No parity
    • 1 stop bit
    • No flow control Connect it to the USART connector
  6. Send the firmware on this second terminal
