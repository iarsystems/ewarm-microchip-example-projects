void ConfigureTc0 (void)

Description
Configure Timer Counter 0 to generate an interrupt every 0.1ms.
This interrupt will be used to display the number of bytes received on the USART.

Configure TC for a 1s (= 1Hz) tick
Variants
Variants Brief description
(void) Configures Timer Counter 0 (TC0) to generate an interrupt every second.
(void) Configure Timer Counter 0 to generate an interrupt every 250ms.
(void) Configure Timer Counter 0 to generate an interrupt every 0.1ms.
