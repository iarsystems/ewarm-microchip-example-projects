basic-twi-slave-project
 
Purpose
This project demonstrates the TWI peripheral in slave mode. It mimics the behavior of a serial memory, enabling the TWI master to read and write data in its internal SRAM.

Requirements
In addition, another device will be needed to act as the TWI master. The basic-twi-eeprom-project can be used for that, in which case a second kit supported by that project is needed. Note: For AT91SAM9261-EK, the TWI slave is available on AT91SAM9261S chip.

Description
After launching the program, the device will act as a simple TWI-enabled serial memory containing 512 bytes. This enables this project to be used with the basic-twi-eeprom-project as the master.

To write in the memory, the TWI master must address the device first, then send two bytes containing the memory address to access. Additional bytes are treated as the data to write.

Reading is done in the same fashion, except that after receiving the memory address, the device will start outputting data until a STOP condition is sent by the master.

Usage
  1. The default address for the TWI slave is fixed to Ox50. If the board has a TWI component with this adress, you can change the define AT24C_ADDRESS in main.c of twi-eeprom project, and the define SLAVE_ADDRESS in main.c of twi-slave project.

  1. For AT91CAP9-DK board:
  2. Connect TWD (SDA) for the 2 boards: pin 6 of connector J18
  3. Connect TWCK(SCL) for the 2 boards: pin 5 of connector J18
  4. Connect GND for the 2 boards: pin 34 of connector J18
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 36 of J18 is 3,3V)

  1. For AT91SAM7L-EK board:
  2. Connect TWD (SDA) for the 2 boards: pin 12 of connector PIOC
  3. Connect TWCK(SCL) for the 2 boards: pin 11 of connector PIOC
  4. Connect GND for the 2 boards: pin 34 of connector PIOC
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 36 of PIOC is 3,3V)

  1. For AT91SAM7SE-EK board:
  2. Connect TWD (SDA) for the 2 boards: pin 3 of connector J15
  3. Connect TWCK(SCL) for the 2 boards: pin 6 of connector J15
  4. Connect GND for the 2 boards: pin 38 of connector J15
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 36 of J15 is 3,3V)

  1. For AT91SAM9260-EK board:
  2. Connect TWD (SDA) for the 2 boards: pin 23 of connector J23
  3. Connect TWCK(SCL) for the 2 boards: pin 26 of connector J23
  4. Connect GND for the 2 boards: pin 36 of connector J23
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 34 of J23 is 3,3V)

  1. For AT91SAM9G20-EK board:
  2. Connect TWD (SDA) for the 2 boards: pin 23 of connector J23
  3. Connect TWCK(SCL) for the 2 boards: pin 26 of connector J23
  4. Connect GND for the 2 boards: pin 36 of connector J23
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 34 of J23 is 3,3V)

  1. For AT91SAM9M10-EK board:
  2. No pin accessible, user should solder wires.

  1. For AT91SAM9RL-EK board:
  2. Connect TWD (SDA) for the 2 boards: pin 23 of connector J23
  3. Connect TWCK(SCL) for the 2 boards: pin 26 of connector J23
  4. Connect GND for the 2 boards: pin 36 of connector J23
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 34 of J23 is 3,3V)

  1. For AT91SAM9XE-EK board:
  2. Connect TWD (SDA) for the 2 boards: pin 23 of connector J23
  3. Connect TWCK(SCL) for the 2 boards: pin 26 of connector J23
  4. Connect GND for the 2 boards: pin 36 of connector J23
  5. Add a pull up of 2,2KOhms on TWD and TWCK (pin 34 of J23 is 3,3V)

  1. For the TWI Slave board:
  2. Compile the application.
  3. Connect the DBGU port of the evaluation board to the computer and open it in a terminal.
    • Settings: 115200 bauds, 8 bits, 1 stop bit, no parity, no flow control.
  4. Download the program inside the evaluation board and run it. Please refer to the SAM-BA User Guide, the GNU-Based Software Development application note or to the IAR EWARM User Guide, depending on your chosen solution.
  5. Upon startup, the application will output the following line on the DBGU: 
    -- Basic TWI Slave Project xxx --
-- AT91xxxxxx-xx
-- Compiled: xxx xx xxxx xx:xx:xx --
-I- Configuring the TWI in slave mode
  6. For the TWI Master board, see the description inside his project
  7. and the "Master" board will output:
