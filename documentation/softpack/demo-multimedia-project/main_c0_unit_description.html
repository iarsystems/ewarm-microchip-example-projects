Description
Purpose
This file contains all the specific code for the demo-multimedia-project

Contents
The demo-multimedia-project application can be roughly broken down as follows:
  • SD Card initialization (MCI configuration + FAT file system)
  • LCD initialization
  • Touchscreen initialization and calibration
  • Read the list of the pictures contained in the root directory of the SD Card
  • Display the first image of the list.
  • In interrupt, the events Pen Pressed and Pen Released are managed thanks to the callbacks functions TSD_PenPressed and TSD_PenReleased (called in interrupt). These callback modify the global variable imageToDisplay.
  • In the main infinite loop, the current image index is compared with the value of imageToDisplay. If different, the new picture is displayed on LCD screen.
