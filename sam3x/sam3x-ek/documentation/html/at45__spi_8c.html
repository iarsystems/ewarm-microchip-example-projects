)

Sends a command to the dataflash through the SPI. The command is identified by its command code and the number of bytes to transfer (1 + number of address bytes + number of dummy bytes).If data needs to be received, then a data buffer must be provided.

Note:
This function does not block; its optional callback will be invoked when the transfer completes.
Parameters:
pAt45Pointer to the At45 instance to initialize.
ucCmdCommand code.
ucCmdSizeSize of command code + address bytes + dummy bytes.
pucDataData buffer.
dwDataSizeNumber of data bytes to send/receive.
dwAddressAddress at which the command is performed if meaningful.
pCallbackOptional callback to invoke at end of transfer.
pArgumentOptional parameter to the callback function.
Returns:
0.
