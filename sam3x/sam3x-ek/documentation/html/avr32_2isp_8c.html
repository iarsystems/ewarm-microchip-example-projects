#define POLYNOMIAL   (ISP_CFG1_CRC8_POLYNOMIAL << 7)

Calculates the CRC-8-CCITT.

CRC-8-CCITT is defined to be x^8 + x^2 + x + 1

To use this function use the following template:

crc = Crc8( crc, data );
