void LowLevelInit ( void   )

Performs the low-level initialization of the chip.

Performs the low-level initialization of the chip. This includes EFC and master clock configuration. It also enable a low level on the pin NRST triggers a user reset.

Note:
We start the chip at 24MHz, in order to reduce voltage lose in case an amperemeter is plugged on the board instead of the VDDCORE jumper. If the voltage lose more than 0.18V, the voltage on VDDCORE will less than 1.62V (1.80 - 0.18 = 1.62), then we will fail to boot the chip.
