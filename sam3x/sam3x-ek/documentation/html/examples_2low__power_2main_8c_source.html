00030 /**
00031  * \page low_power Lowpower Example
00032  *
00033  * \section Purpose
00034  * This example allows to measure the consumption of the core in different modes
00035  * (active mode, sleep mode, wait mode, backup mode).
00036  *
00037  * \section Requirements
00038  *
00039  * This package can be used with SAM3X evaluation kits.
00040  *
00041  * \section Description
00042  *
00043  * At startup, the program configures all the PIOs as input to avoid parasite
00044  * consumption (except the UART PIOs and push button PIO). Then a menu is displayed.
00045  * It is divided into 2 parts. First part allows user to change the configuration
00046  * (MCK/PCK frequency configuration, flash access mode etc.),
00047  * The second part allows user to enter in a mode To measure consumption.
00048  * An amperemeter has to be plugged on the board instead of the VDDIN jumper.
00049  *
00050  *
00051  * \section Usage
00052  *
00053  * -# Build the program and download it inside the evaluation board. Please
00054  *    refer to the
00055  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">
00056  *    SAM-BA User Guide</a>, the
00057  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00058  *    GNU-Based Software Development</a> application note or to the
00059  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00060  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00061  * -# On the computer, open and configure a terminal application
00062  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00063  *   - 19200 bauds
00064  *   - 8 bits of data
00065  *   - No parity
00066  *   - 1 stop bit
00067  *   - No flow control
00068  * -# Start the application.
00069  * -# In the terminal window, the following text should appear:
00070  *    \code
00071  *     -- Low_power Example xxx --
00072  *     -- xxxxxx-xx
00073  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00074  *    \endcode
00075  * -# Press one of the keys listed in the menu to perform the corresponding action.
00076  *
00077  * \section References
00078  * - low_power/main.c

00592 /**
00593  * \brief Performs the low-level initialization of the chip.
00594  *
00595  * \note We start the chip at 24MHz, in order to reduce voltage lose in case
00596  *       an amperemeter is plugged on the board instead of the VDDCORE jumper.
00597  *       If the voltage lose more than 0.18V, the voltage on VDDCORE will less
00598  *       than 1.62V (1.80 - 0.18 = 1.62), then we will fail to boot the chip.
00599  */
00600 extern void LowLevelInit( void )
00601 {
00602     uint32_t dwTimeout;
00603 
00604     /* Set 3 WS for Embedded Flash Access */
00605     EFC0->EEFC_FMR = EEFC_FMR_FWS( 3 );
00606     EFC1->EEFC_FMR = EEFC_FMR_FWS( 3 );
00607 
00608     /* Initialize main oscillator */
00609     if ( !(PMC->CKGR_MOR & CKGR_MOR_MOSCSEL) )  /* Main Oscillator Selection */
00610     {
