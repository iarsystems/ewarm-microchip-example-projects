00030 /**
00031  * \page spi_dma_slave SPI Slave Example
00032  *
00033  * \section Purpose
00034  *
00035  * This example uses Serial Peripheral Interface (SPI1) in slave mode to
00036  * communicate to another interface (SPI0) in master mode .
00037  *
00038  * \section Requirements
00039  *
00040  * This package can be used with  at91sam3x-ek board by switch off the data flash.
00041  * Please connect the SPI pins :
00042  *        - <b>SPI0(MASTER)--SPI1(SLAVE)</b>
00043  *        - SPI0_NPCS0(PA28)--SPI1_NPCS0(PE31)
00044  *        - SPI1_MISO(PA25)--SPI1_MISO(PAE28)
00045  *        - SPI1_MOSI(PA26)--SPI1_MOSI(PE29)
00046  *        - SPI0_SPCK(PA27)--SPI1_SPCK(PE30)
00047  *
00048  * \section Descriptions
00049  *
00050  * This example shows control of the SPI slave, how to configure and
00051  * transfer data with SPI slave. By default, example runs in SPI slave mode,
00052  * waiting SPI slave & UART inputs.
00053 
00054  * The code can be roughly broken down as follows:
00055  * <ul>
00056  * <li> 't' will start SPI transfer test
00057  * <ol>
00058  * <li>Configure SPI as master, setup SPI clock.
00059  * <li>Send 4 byte CMD_TEST to indicate the start of test.
00060  * <li>Send several 64 bytes block, on transmitting next block, the content of last block is returned.
00061  * <li>Send CMD_STATUS command and waiting for the status reprots from slave.
00062  * <li>Send CMD_END command to indicate the end of test.
00063  * </ol>
00064  * <li>Setup SPI clock for master.
00065  * </ul>
00066  *
00067  * \section Usage
00068  *
00069  * -# Compile the application.
00070  * -# Connect the DBGU port of the evaluation board to the computer and open
00071  * it in a terminal.
00072  *    - Settings: 115200 bauds, 8 bits, 1 stop bit, no parity, no flow control.
00073  * -# Download the program inside the evaluation board and run it. Please refer to
00074  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">
00075  *    SAM-BA User Guide</a>, the
00076  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00077  *    GNU-Based Software Development</a> application note or to the
00078  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00079  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00080  * -# Upon startup, the application will output the following line on the DBGU:
00081  *    \code
00082  *     -- Spi slave Example  xxx --
00083  *     -- AT91xxxxxx-xx
00084  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00085  *    \endcode
00086  * -# The following traces detail operations on the spl slave example, displaying success
00087  *    or error messages depending on the results of the commands.
00088  *
00089  * \section References
00090  * - spi_dma_slave/main.c
00091  * - spi.c

00280

00282                 /* Only CMD_DATA accepted*/

00284                 {
00285                     ucSpi1State = SLAVE_STATE_DATA ;

00288             break;
00289

00291                 spi1Status.totalNumBlocks++ ;

00293                 if ( spi1Status.totalNumBlocks == ucNumTestBlocks )
00294                 {
00295                     ucSpi1State = SLAVE_STATE_STATUS_ENTRY;
00296                 }
00297             break ;

00300                 ucSpi1State = SLAVE_STATE_STATUS ;
00301             break ;
