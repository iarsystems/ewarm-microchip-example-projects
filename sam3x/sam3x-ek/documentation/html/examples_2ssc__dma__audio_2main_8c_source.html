00030 /**
00031  * \page ssc_dma_audio SSC with DMA Audio Example
00032  *
00033  * \section Purpose
00034  * This example uses the Synchronous Serial Controller (SSC) of an AT91SAM3x microcontroller
00035  * to output an audio steam through the on-board WM8731 CODEC.
00036  *
00037  * \section Requirements
00038  *
00039  * This package can be used with AT91SAM3x evaluation kits with external codec WM8731 components.
00040  *
00041  * \section Description
00042  * This program plays a WAV file pre-loaded into the flash. The audio stream is sent through
00043  * the SSC interface connected to the on-board WM8731, enabling the sound to be audible
00044  * using a pair of headphones.
00045  *
00046  * Since the WM8731 DAC requires that it be feeded a master clock multiple of the sample rate,
00047  * it is difficult to handle any WAV file. As such, this example application is limited to
00048  * playing files with the following format:
00049  * - Format: WAV
00050  * - Sample rate: 48 kHz
00051  *
00052  * The code can be roughly broken down as follows:
00053  * <ul>
00054  *   <li> Enable the clock </li>
00055  *   <li> Load WAV file information</li>
00056  *   <li> Configure and enable the Codec</li>
00057  *   <li> Configure and enable the SSC interrupt</li>
00058  *   <li> Play WAV file</li>
00059  * </ul>
00060  *
00061  * \section usage
00062  * -# Build the program and download it inside the evaluation board. Please
00063  *    refer to the
00064  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/6421B.pdf">
00065  *    SAM-BA User Guide</a>, the
00066  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00067  *    GNU-Based Software Development</a> application note or to the
00068  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00069  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00070  * -# On the computer, open and configure a terminal application
00071  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00072  *   - 115200 bauds
00073  *   - 8 bits of data
00074  *   - No parity
00075  *   - 1 stop bit
00076  *   - No flow control
00077  * -# Start the application.
00078  * -# In the terminal window, the following text should appear:
00079  *    \code
00080  *     -- SSC DMA audio example xxx --
00081  *     -- SAMxx-xx
00082  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00083  *     Menu :
00084  *     ------
00085  *     W: Play the WAV file pre-loaded in flash
00086  *     I: Display the information of the WAV file
00087  *     X: Receive WAV file through DBGU
00088  *    \endcode
00089  * The user can then choose any of the available options to perform the described action.
00090  *
00091  * \section References
00092  * - ssc_dma_audio/main.c
00093  * - ssc.c
00094  * - twi.c
00095  * - twid.c
00096  * - dmac.c
00097  * - dmad.c
00098  * - wm8731.c 
00099  */
