Detailed Description

Implement EMAC data transfer and PHY management functions.

Usage

  1. Implement EMAC interrupt handler, which must invoke EMACD_Handler() to handle EMAC interrupt events.
  2. Implement sEmacd instance in application.
  3. Initialize the instance with EMACD_Init() and EMACD_InitTransfer(), so that EMAC data can be transmitted/received.
  4. Some management callbacks can be set by EMACD_SetRxCallback() and EMACD_SetTxWakeupCallback().
  5. Send ethernet packets using EMACD_Send(), EMACD_TxLoad() is used to check the free space in TX queue.
  6. Check and obtain received ethernet packets via EMACD_Poll().
