00053  * \defgroup udc_group USB Device Controller (UDC)

00055  * The UDC provides a high-level abstraction of the usb device.

00064  * Vendor ID provided by USB org (ATMEL 0x03EB).
00065  * 
00066  * USB_DEVICE_PRODUCT_ID (Word)<br>
00067  * Product ID (Referenced in usb_atmel.h).
00068  * 
00069  * USB_DEVICE_MAJOR_VERSION (Byte)<br>
00070  * Major version of the device
00071  *
00072  * USB_DEVICE_MINOR_VERSION (Byte)<br>
00073  * Minor version of the device
00074  * 
00075  * USB_DEVICE_MANUFACTURE_NAME (string)<br>
00076  * ASCII name for the manufacture
00077  * 
00078  * USB_DEVICE_PRODUCT_NAME (string)<br>
00079  * ASCII name for the product
00080  * 
00081  * USB_DEVICE_SERIAL_NAME (string)<br>
00082  * ASCII name to enable and set a serial number
00083  * 
00084  * USB_DEVICE_POWER (Numeric)<br>
00085  * (unit mA) Maximum device power
00086  * 
00087  * USB_DEVICE_ATTR (Byte)<br>
00088  * USB attributes available:
00089  *  - USB_CONFIG_ATTR_SELF_POWERED

00091  *  Note: if remote wake enabled then defines remotewakeup callbacks,

00093  *  
00094  * USB_DEVICE_LOW_SPEED (Only defined)<br>
00095  * Force the USB Device to run in low speed
00096  * 
00097  * USB_DEVICE_HS_SUPPORT (Only defined)<br>
00098  * Authorize the USB Device to run in high speed
00099  * 
00100  * USB_DEVICE_MAX_EP (Byte)<br>
00101  * Define the maximum endpoint number used by the USB Device.<br>
00102  * This one is already defined in UDI default configuration. 
00103  * Ex:
00104  * - When endpoint control 0x00, endpoint 0x01 and
00105  *   endpoint 0x82 is used then USB_DEVICE_MAX_EP=2
00106  * - When only endpoint control 0x00 is used then USB_DEVICE_MAX_EP=0
00107  * - When endpoint 0x01 and endpoint 0x81 is used then USB_DEVICE_MAX_EP=1<br>

00131  *     // Authorize VBUS monitoring \n

00133  *       // Implement custom VBUS monitoring via GPIO or other \n
00134  *     } \n

00136  *     { \n

00138  *        udc_attach(); \n

00140  *

00142  *     Event VBUS present() // VBUS interrupt or GPIO interrupt or .. \n

00145  *     } \n
