dac12_pdc_sinewave

This application is aimed to demonstrate how to use DACC in free running mode. The example allows to configure the frequency and amplitude of output sinewave. The frequency could be set from 200Hz to 3KHz, and the peak amplitude could be set from 100 to 2047 in regard to 12bit resolution. The output could be monitored by connecting PB16/DAC0 to one channel of an oscilloscope.
