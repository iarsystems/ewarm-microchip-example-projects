pio_alternate_function

The application shows the alternate function of ERASE pin, which extends PIO from a dedicated system I/O line. It first programs the internal flash with a walking pattern and then enable ERASE pin in PIO mode. After the erase operation, it check the content of flash, which is unchanged due to ERASE in PIO mode. In the next step, it put ERASE pin in system I/O mode,which will enable the erase function. The content of flash should be erased to 0xFF, which indicates the ERASE pin functioning. The pins which are used by USB(DDM,DDP),JTAG(TCK,TMS,TDO,TDI) and Oscillator(XIN, XOUT,XIN32,XOUT32) could also be alternated between general purpose I/O and system I/O in the similar way.
