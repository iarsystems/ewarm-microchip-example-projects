Example description

Clock system example 3

DESCRIPTION
===========
  This example shows how to switch between various system clock sources
  and prescalers at run time.

  The Systick is used to generate periodic interrupts based on its CPU clock.
  With each interrupt, a LED (LED_EXAMPLE_OUTPUT) is toggled,
  thus giving a visual indication of the system clock frequency.

  The clock system is reconfigured every time the user presses a button
  (NEXT_BUTTON) to use the next clock source and prescaler settings.

  Main files:
  - clock_example3_sam4l.c: clock system example application
  - conf_board.h: board initialization configuration
  - conf_clock.h: system clock configuration
  - conf_example3.h: chip- or board-specific example configuration

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Download and run the example.
