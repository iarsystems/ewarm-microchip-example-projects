Example description

CPU reset cause example for SAM4L

DESCRIPTION
===========
  This example demonstrates how to use the reset cause interface.

  The program turns LED0 on when the power-on reset happens and outputs
  different informaiton on the terminal, depending on the reset cause.

  Type different commands to generate different reset. To generate Core BOD 
  reset or I/O BOD reset, press reset button after typing 'c' or 'i'. 

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART port via a terminal program on your PC.
  4) Download and run the example.
