Example description

Common IOPORT service example 2

DESCRIPTION
===========
  This example demonstrates how to use the common IOPORT service for pin
  control.

  This example flashes a single LEDs present with flash speed of 1Hz.
 
  Additionally a pin input can inspected from a debug session.

  Main Files
   - ioport.h common gpio definitions
   - ioport_example2.c example application
   - conf_example.h example definitions

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Download and run the example.
