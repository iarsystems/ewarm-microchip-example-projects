Example description

USB host mass storage

DESCRIPTION
===========
  This example shows how to implement a USB host mass storage
  on SAM4L MCU.

  After loading firmware, connect the board to a U-Disk
  (FAT/FAT32 are supported). This example creates a file "uhi_msc_test.txt"
  on all present U-disks.

  The example uses the following module groups:
  - Basic modules:
    Startup, board, clock, interrupt, power management
  - USB host stack and MSC modules:
      services/usb/
      services/usb/uhc/
      services/usb/class/msc/host/
  - Thirdparty modules:
      thirdparty/fatfs
  - Specific implementation:
     - main.c,
           initializes clock
           initializes interrupt
           manages UI
     - specific implementation for each target "./examples/product_board/":
        - conf_foo.h   configuration of each module
        - ui.c        implement of user's interface (buttons, leds)

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory
            with no debug info.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Download and run the example.
