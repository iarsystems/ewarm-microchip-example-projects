SAM4L-EK Getting Started Application

DESCRIPTION
===========
  The Getting Started example will help new users get familiar with Microchip's
  SAM family of microcontrollers. This basic application shows the startup
  sequence of a chip and how to use its core peripherals.
 
  The demonstration program makes the LED(s) on the board blink at a fixed rate.
  This rate is generated by using Time tick timer. The blinking can be stopped
  using the push button.

  Configure system tick to get 1ms tick period.
  Configure button.
  Configure TC.
  Press PB0 to Start/Stop LED0 blinking.

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART port via a terminal program on your PC.
  4) Download and run the example.
