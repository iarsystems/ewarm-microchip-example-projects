Example description

LCD Glass component example

DESCRIPTION
===========
  This example demonstrates how to use the C42364A LCD Glass driver with the
  32 KHz oscillator crystal.
  
  Main Files
  - c42364a_lcdca.c: C42364A LCD Glass component driver using LCDCA;
  - c42364a_slcdc.c: C42364A LCD Glass component driver using SLCDC;
  - c42364a.h: C42364A LCD Glass component driver header file;
  - c42364a_example.c: C42364A LCD Glass component example application.
 
COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
