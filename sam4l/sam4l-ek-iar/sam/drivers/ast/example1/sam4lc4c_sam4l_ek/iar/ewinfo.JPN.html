Example description

Asynchronous Timer (AST) example 1 (calendar mode)

DESCRIPTION
===========
  The ast_example1 example demonstrates how to use the asynchronous timer.
  Upon startup, the program uses the USART driver to display a real time
  counter incremented every second.
  The display is triggered by the interrupt controller module.

  Main Files
  - ast.c: Asynchronous Timer driver
  - ast.h: Asynchronous Timer driver header file
  - ast_example1.c: Asynchronous Timer example application

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
