Example description

Asynchronous Timer (AST) Example 2 (alarm wakeup)

DESCRIPTION
===========
  This example demonstrates how to use the AST driver with the 32kHz
  oscillator crystal and to also setup an alarm to wake-up the device from its low
  power modes.

  Upon startup, the program uses the USART driver to display a menu that lets
  the user select the low power mode from which the device will wake-up when
  the AST Alarm triggers.

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
