CRCCU Example 

DESCRIPTION
===========
  This example demonstrates the Cyclic Redundancy Check Calculation Unit (CRCCU)
  provided on SAM series of microcontrollers. It shows how to use CRCCU
  to compute CRC on a memory area. 

  The example work as:
  a. Initialize a buffer in SRAM with random data;
     Initialize a buffer in Flash with the same data as in SRAM.
  b. Initialize the CRCCU to compute the CRC of the buffer in SRAM
     with one of the 3 CRC algorithms.
  c. Start the computation.
  d. Read the result and display it.
  e. Calculate and display the CRC result in Flash and compare it with
     the result in SRAM .
  f. Redo b, c, d, e with 2 other CRC modes

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
