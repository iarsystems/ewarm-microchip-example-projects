Example description

DAC Sinewave Example 

DESCRIPTION
===========
  The DAC Sinewave example demonstrates how to use DACC peripheral.  

  This application is aimed to demonstrate how to use DACC in free running
  mode.

  The example allows to configure the frequency and amplitude of output
  sinewave. The frequency could be set from 200Hz to 3KHz, and the peak
  amplitude could be set from 100 to 1023/4095 in regard to 10/12bit
  resolution.

  The example can also generate a full amplitude square wave for reference.
 
  The output could be monitored by connecting the DAC out (PA06) PIN, that is
  used to a channel of the oscilloscope. 

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
