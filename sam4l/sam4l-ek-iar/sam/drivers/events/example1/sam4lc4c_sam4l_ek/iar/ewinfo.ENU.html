Example description

Peripheral Event Controller (PEVC) example 1.

DESCRIPTION
===========
  This example shows how to use the Peripheral Event Controller.
  In the example, the AST generates a periodic event which is transmitted
  to the PDCA. Each time a new event occurs, a character is sent to the
  USART without the use of the CPU. The main loop of the function is a
  delay 500ms and toggle a LED continuously to show CPU activity.

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
