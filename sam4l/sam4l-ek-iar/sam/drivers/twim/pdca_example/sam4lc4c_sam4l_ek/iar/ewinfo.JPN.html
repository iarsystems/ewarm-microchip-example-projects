Example description

TWIM PDCA Example for SAM. 

DESCRIPTION
===========
  This application gives an example of how to use TWIM driver of SAM to
  access an TWI-compatible EEPROM via PDCA.

  The program needs a TWI-compatible EEPROM connected with the TWIM module.

  At first, the specified TWIM write some data pattern to the EEPROM, then
  read it back and check if the written and the read match in the end.

COMPATIBILITY
=============
  The example project is compatible with Microchip SAM4L-EK evaluation board.

CONFIGURATION
=============
  Debug   - The configuration runs the example from the internal flash memory.
  Release - The configuration runs the example from the internal flash memory 
            with no debug info.

  Please set the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

  Use the virtual JLink CDC UART Port number, when configuring the COM port.

GETTING STARTED
===============
  1) Choose a confguration.
  2) Build the example.
  3) Open the JLink CDC UART Port with a terminal program on your PC.
  4) Download and run the example.
