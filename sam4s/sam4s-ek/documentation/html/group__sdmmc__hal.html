Detailed Description

Purpose

Implement MultiMediaCard(MCI) Interface Driver

Usage

  1. MCI_Init(): Initializes a MCI driver instance and the underlying peripheral.
  2. MCI_Handler() : Interrupt handler which is called by ISR handler.
  3. MCI_SetSpeed() : Configure the MCI CLKDIV in the _MR register (Hsmci::HSMCI_MR).
  4. MCI_SetBusWidth() : Configure the MCI SDCBUS in the _SDCR register (Hsmci::HSMCI_SDCR).
  5. MCI_EnableHsMode() : Configure the MCI HSMODE in the _CFG register (Hsmci::HSMCI_CFG).
