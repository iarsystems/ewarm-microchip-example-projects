usart_serial

On start up, the debug information is dumpped to on-board UART port. A terminal application, such as hyperterminal, is used to monitor these debug information. Open another hyperterminal to connect with on-board USART port. Then the program works in ECHO mode, so USART will sends back anything it receives from the hyperterminal. You can send a text file from the hyperterminal connected with USART port to the device (without any protocol such as X-modem).
