ek-demo

SAM4S-EK2 Demo Example

DESCRIPTION
===========

  The ek-demo will help new users get familiar with Microchip's SAM4S family of
  microcontrollers and SAM4S-EK2 development board. This demo shows the startup
  sequence of a chip and how to use its core peripherals. It is based on FreeRTOS
  operating system.

COMPATIBILITY
=============

  The example project is compatible with Microchip SAM4S-EK2 evaluation board.

CONFIGURATION
=============

  ek-demo - flash - The configuration runs the example from the internal flash memory.

  If you connect the PC to the board's UART connector via serial cable, please set
  the following settings in your terminal software:

  Baud rate : 115200
  Data bits : 8
  Parity    : none
  Stop bits : 1

GETTING STARTED
===============

  1) Choose "ek-demo - flash" confguration.

  2) Open Batch Build window (press F8) and build the "ek-demo" batch configuration.
     This will build all required libraries.

  3) Download and run the example.
